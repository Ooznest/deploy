1) Connect USB-C Cable

Connect USB-C Cable to your computer. Once connected, the next step is to choose one bootloader path below, depending on the state of the controller firmware.

Next: Choose a bootloader path below

2) Manual Bootloader Entry

Use this for controllers with no firmware or corrupted firmware. Then click Connect Device.

Press and hold the MODE button,

then click RESET, then release MODE.

Click Connect Device.

2) Automatic Bootloader Entry

Use this for controllers already running grblHAL v20260622 or newer.

Click Enter Bootloader.

4) Flash grblHAL

Once one of the step 3 bootloader actions has completed, click Flash grblHAL to deploy the firmware.

Firmware installation complete

Either press RESET on the controller to reboot the device with the new firmware, or power cycle the controller.

To power cycle fully: power off, remove the USB cable so the power provided by the USB cable is removed as well, then reconnect USB and power the controller up again.

The controller must be reset or power cycled for a fresh boot using the new firmware.

After that, connect to it using Ooznest Control.
